# Active Directory Homelab

## Project Goal

Building a Windows Server environment with Active Directory to improve my skills and practical knowledge in IT system administration.

## Technologies Used

- Windows Server 2025
- Windows 11
- Active Directory Domain Services (AD DS)
- DNS
- DHCP
- Group Policy (GPO)
- SMB File Shares
- PowerShell
- Oracle VirtualBox

## Implemented Features

- Active Directory Domain Services (AD DS)
- DNS Server
- DHCP Server
- User Management
- Group Management
- Organizational Units (OUs)
- SMB File Shares
- Group Policy Objects (GPOs)
- Domain Join of Windows Clients
- PowerShell Administration
